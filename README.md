## 🚀 Quick Start

1. **Install Dependencies:**
```bash
pip install -r requirements.txt
```

2. **Configure API Keys:**
Edit `config/api_keys.json` with your credentials:
- Higgsfield.ai API
- OpenAI/Claude API
- Platform credentials
- Payment processors

3. **Launch System:**
```bash
python src/main.py
```
